Feedstock for [lhapdf](https://lhapdf.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

LHAPDF is the standard tool for evaluating parton distribution functions (PDFs) in high-energy physics.

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `6.5.6` |
| linux-arm64 (`linux-aarch64`) | ✅ `6.5.6` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge lhapdf
```
