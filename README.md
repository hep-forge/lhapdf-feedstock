Feedstock for [lhapdf](https://lhapdf.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).

LHAPDF is the standard tool for evaluating parton distribution functions (PDFs) in high-energy physics.

## Install

```bash
conda install -c hep-forge -c conda-forge lhapdf
```
